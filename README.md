## Controls ##

Move Player       =  <Left/Right Arrow>
Climb Ladder      =  <Up/Down Arrow>
Jump              =  <Spacebar>

Pick Up Item      =  <Left CTRL>
Drop Item         =  <Left ALT>
Use Item          =  <Left SHIFT>

Restart Level     =  <R>
Return to Title   =  <Backspace>

Next Level Cheat  =  <Plus/Equals Key>
